# File Organizer

## Process
1.  **Inventory**: List all files and identify duplicates or temporary files.
2.  **Categorization**: Group files based on their purpose (src, docs, assets).
3.  **Renaming**: Apply consistent naming conventions.
4.  **Relocation**: Move files to their designated directories.
