# MCP Server Development Guide

## Workflow
1.  **Phase 1: Research and Planning**: Understand the target API and design the tools.
2.  **Phase 2: Implementation**: Set up the project and implement core logic.
3.  **Phase 3: Review and Test**: Use the MCP Inspector to verify tool behavior.
4.  **Phase 4: Create Evaluations**: Create Q&A pairs to test LLM effectiveness.
