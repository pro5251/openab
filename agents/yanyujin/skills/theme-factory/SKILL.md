# Theme Factory

## Process
1.  **Moodboard**: Identify the core emotions and tone (e.g., "professional," "playful," "brutalist").
2.  **Color Palette**: Define primary, secondary, accent, and background colors using HSL or CSS variables.
3.  **Typography Scale**: Select font families and define a clear hierarchy (H1-H6, body, caption).
4.  **Component Styling**: Define standard borders, shadows, and spacing (gap, padding).
5.  **Implementation**: Create a `theme.css` or equivalent file with the defined variables.
