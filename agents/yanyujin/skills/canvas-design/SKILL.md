# Canvas Design

## Process
1.  **Coordinate System**: Define the canvas grid and how elements are positioned.
2.  **Rendering Loop**: Implement a `requestAnimationFrame` loop for smooth animations.
3.  **Interaction**: Map mouse/touch events to canvas coordinates and trigger updates.
4.  **Optimization**: Use techniques like off-screen canvases or spatial partitioning for performance.
