# PPTX Data Processing

## Tools
*   `pptxgenjs` (Node.js) or `python-pptx` (Python)

## Process
1.  **Template Loading**: Load an existing PPTX as a base.
2.  **Slide Generation**: Add new slides with specific layouts.
3.  **Content Population**: Insert text, tables, images, and charts.
4.  **Styling**: Apply themes and master slide properties.
