# PDF Data Processing

## Tools
*   `pdf-lib` / `pdfkit` (Node.js) or `PyPDF2` / `reportlab` (Python)

## Process
1.  **Parsing**: Extract text or data from existing PDF documents.
2.  **Generation**: Create new PDFs with custom layouts, text, and images.
3.  **Manipulation**: Merge, split, or rotate PDF pages.
4.  **Security**: Apply passwords or digital signatures.
