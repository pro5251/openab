# Changelog Generator

## Process
1.  **Analyze Git History**: Get a list of commits since the last tag/release.
2.  **Categorize Commits**: Group commits by type (Feat, Fix, Refactor, Docs).
3.  **Format Output**: Generate a Markdown formatted list of changes.
4.  **Update CHANGELOG.md**: Append the new entry to the existing file.
