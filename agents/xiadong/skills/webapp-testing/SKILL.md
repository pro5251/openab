# Web Application Testing

To test local web applications, write native Python Playwright scripts.

## Process
1.  **Identify Selectors**: Inspect the rendered DOM to find buttons, links, and inputs.
2.  **Write Automation**: Use Playwright to navigate, interact, and assert.
3.  **Wait for Networkidle**: Ensure the app is fully loaded before interacting.
4.  **Screenshot/Logs**: Capture visual and console feedback for debugging.
