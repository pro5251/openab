# Test-Driven Development

## Workflow
1.  **Planning**: Confirm interface changes and behaviors to test.
2.  **Tracer Bullet**: Write ONE test that fails, then minimal code to pass.
3.  **Incremental Loop**: One test at a time, repeat RED -> GREEN.
4.  **Refactor**: Clean up the code while staying at GREEN.
